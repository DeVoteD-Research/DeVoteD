Metadata

          For the sake of transparency, all datasets, which has been processed during our project, can be accessed either on our GitHub repository or on the respective pages linked below. All metadata is available as .ttl files formed by following the standard DCAT-AP 3.0. The files have been retrieved from Varieties of Democracy (V-Dem), Harvard Dataverse - Party Facts Version 2023 (PF), Voter Turnout Databse - International IDEA (VTB) and Manifesto Project (MAN)
